## 用法

<!-- Control the opacity of an element's background color using the `bg-opacity-{amount}` utilities. -->
使用 `bg-opacity-{amount}` 功能來控制元素背景顏色的透明度。

```html indigo
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="h-16 rounded-md bg-opacity-100 bg-indigo-600 font-extrabold text-white flex justify-center items-center">100%</div>
    <div class="h-16 rounded-md bg-opacity-75 bg-indigo-600 font-extrabold text-white flex justify-center items-center">75%</div>
    <div class="h-16 rounded-md bg-opacity-50 bg-indigo-600 font-extrabold text-white flex justify-center items-center">50%</div>
    <div class="h-16 rounded-md bg-opacity-25 bg-indigo-600 font-extrabold text-white flex justify-center items-center">25%</div>
    <div class="h-16 rounded-md bg-opacity-0 bg-indigo-600 font-extrabold text-white flex justify-center items-center">0%</div>
  </div>
</template>

<div class="bg-indigo-600 **bg-opacity-100** ..."></div>
<div class="bg-indigo-600 **bg-opacity-75** ..."></div>
<div class="bg-indigo-600 **bg-opacity-50** ..."></div>
<div class="bg-indigo-600 **bg-opacity-25** ..."></div>
<div class="bg-indigo-600 **bg-opacity-0** ..."></div>
```

## 響應式

<!-- To control an element's background color opacity at a specific breakpoint, add a `{screen}:` prefix to any existing background color opacity utility. For example, use `md:bg-opacity-50` to apply the `bg-opacity-50` utility at only medium screen sizes and above. -->
如要讓元素的背景顏色透明度有響應式效果，可在現有背景顏色透明度功能前加上前綴詞 `{screen}:`。例如：中型或以上的螢幕尺寸使用 `md:bg-opacity-50` 以套用 `bg-opacity-50` 功能。

```html
<div class="bg-blue-500 bg-opacity-75 **md:bg-opacity-50**">
  <!-- ... -->
</div>
```

## 客製化

<!-- To customize the opacity values for all opacity-related utilities at once, use the `opacity` section of your `tailwind.config.js` theme configuration: -->
使用 `tailwind.config.js` 中的 `opacity` 可一次自訂所有跟透明度相關的值。

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        opacity: {
+         '15': '0.15',
+         '35': '0.35',
+         '65': '0.65',
        }
      }
    }
  }
```

<!-- If you want to customize only the background opacity utilities, use the `backgroundOpacity` section: -->
如果只要自訂背景透明度功能，使用 `backgroundOpacity`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        backgroundOpacity: {
+         '10': '0.1',
+         '20': '0.2',
+         '95': '0.95',
        }
      }
    }
  }
```
