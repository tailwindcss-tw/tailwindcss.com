<!-- ### Configure Tailwind with Laravel Mix -->
### 在 Laravel Mix 中設定 Tailwind

<!-- In your `webpack.mix.js`, add `tailwindcss` as a PostCSS plugin: -->
在你的 `webpack.mix.js` 新增 `tailwindcss` 的 PostCSS 插件：

```diff-js
  // webpack.mix.js
  mix.js("resources/js/app.js", "public/js")
    .postCss("resources/css/app.css", "public/css", [
+     require("tailwindcss"),
    ]);
```

```preval include
tool: Laravel
file: ./resources/css/app.css
```

<!-- Next, import your stylesheet in your main Blade layout (commonly `resources/views/layouts/app.blade.php` or similar) and add the responsive viewport meta tag if it's not already present: -->
再來，在你的主要 Blade layout 引用你的樣式內容 (通常是 `resources/views/layouts/app.blade.php` 或類似檔案) 並且新增響應式 meta tag，如果沒有的話：

```diff-html
  <!doctype html>
  <head>
    <!-- ... --->
+   <meta charset="UTF-8" />
+   <meta name="viewport" content="width=device-width, initial-scale=1.0" />
+   <link href="{{ asset('css/app.css') }}" rel="stylesheet">
  </head>
  <!-- ... --->
```
