---
# title: "Install Tailwind CSS with Vue 3 and Vite"
title: "在 Vite 和 Vue 3 上安裝 Tailwind CSS"
# description: "Setting up Tailwind CSS in a Vue 3 and Vite project."
description: "在 Vite 和 Vue 3 專案上設定 Tailwind CSS。"
# tool: Vue 3 and Vite
tool: Vue3 和 Vite
---

```preval installation
tool: Vite
script: npm init @vitejs/app
npmInstall: true
```

---

```preval setup
version: latest
```

```preval configuration
purge:
  - ./index.html
  - ./src/**/*.{vue,js,ts,jsx,tsx}
```

```preval include
tool: Vite
file: ./src/index.css
create: true
```

<!-- Finally, ensure your CSS file is being imported in your `./src/main.js` file: -->
最後，確認你的 CSS 檔案有被你的 `./src/main.js` 檔案所引用 (import)：

```js
// src/main.js
import { createApp } from 'vue'
import App from './App.vue'
import './index.css'

createApp(App).mount('#app')
```

---

```preval finish
scripts:
  - npm run dev
```
