## 使用方式

透過 `sr-only` 可以在視覺上隱藏元素但不會影響到螢幕報讀機:

```html
<a href="#">
  <svg><!-- ... --></svg>
  <span class="sr-only">Settings</span>
</a>
```

使用 `not-sr-only` 可以復原 `sr-only`, 讓元素可以被使用者跟螢幕報讀機判讀. 這便於當你想要在小螢幕時, 在視覺上的隱藏某個元件, 但是在大螢幕要顯示的情況, 舉個例子:

```html
<a href="#">
  <svg><!-- ... --></svg>
  <span class="sr-only sm:not-sr-only">Settings</span>
</a>
```

預設情況下, `responsive` 跟 `focus` 會生成這個工具. 你可以透過使用 `focus:not-sr-only` 讓元素在預設的情況隱藏, 但在使用者tab到該元素時顯示 - 尤其利於 "前往内容" 連結:

```html
<a href="#" class="sr-only focus:not-sr-only">
  Skip to content
</a>
```
